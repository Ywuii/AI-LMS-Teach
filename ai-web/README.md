<h1>智慧学习管理系统 AI-LMS</h1>

## 介绍

AI-LMS使用Vue3作为前端、Django(DRF)作为后端。

## 快速开始

后端

1. 创建python虚拟环境，建议3.12.9版本
2. pip install -r requirements.txt 安装依赖（本项目选择requirements-ailms.txt）
3. 修改.env文件配置数据库连接
4. 按照数据库配置，创建数据库
5. 运行dbinit.py初始化数据库表
6. 运行后端项目

前端

1. 安装node.js 建议选择最新LTS版本
2. npm install -g pnpm
3. pnpm install
4. pnpm dev
